# 食材価格チェック

農林水産省の「食品価格動向調査」をもとに、野菜・肉・卵の全国平均小売価格を見やすく確認する静的サイトです。

## 使い方

`index.html` を静的ホスティングに置くだけで公開できます。GitHub Pages、Netlify、Vercel などで配信できます。

ローカル確認:

```powershell
python -m http.server 4174 --bind 127.0.0.1
```

## データ更新

農林水産省の最新公表ファイルから `script.js` の価格データを更新します。

```powershell
python scripts/update-data.py
```

このスクリプトは以下を取得します。

- 野菜: 食品価格動向調査（野菜）の Excel
- 食肉・鶏卵: 食品価格動向調査（食肉・鶏卵）の PDF

## 注意

表示価格は全国平均の小売価格です。地域別・店舗別の実売価格ではありません。
「買い時」「様子見」などの判定は、平年比と直近推移から作ったこのサイト独自の目安です。
